### 备注

mooc的作业提交系统有时候很**操蛋**，如果有一个检验始终运行不通过（而你运行起来没问题），可以试着检查以下内容：
- **输出输入格式**是否正确，比如说是否有空格、是否有汉字（个人所得税那道题需要写汉字）等等；
- 检查**数据类型**，有时候必须要使用 `long long` 或者其它特定格式，一般题目里有提示，只不过隐藏得很深；
- 运行时间过长就试着修改你的循环，改一下循环条件或者利用别的方法做简化，减少时间复杂度；
- （你的程序始终改不对）看看有没有混淆赋值 `=` 和等于 `==`；

*我建议你在编译器里面写好了再copy到系统里！*

作业提交系统无法复制任何文字（Ctrl+V和右键都不让用），你可以下载豆包，豆包有悬浮窗辅助功能，当你框选了网页的文字时，豆包会跳出一个悬浮窗，里面有复制键。
